# Spring security
## Spring Security란?
스프링 기반 애플리케이션의 보안을 담당하는 프레임워크로 로그인, 세션 관리, 권한 체크 등등  
보안과 관련된 기본적인 뼈대를 제공한다.  
[스프링 시큐리티](https://docs.spring.io/spring-security/reference/servlet/architecture.html)
### 핵심요소 
1. 인증 (Authentication)
2. 인가 (Authorization)
3. 악의적인 공격 방어 (Protection Against Exploits)
